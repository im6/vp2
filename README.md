## QuickStart

```sh
yarn install
npm start
```

local dev django server

```sh
pip3 install virtualenv
virtualenv -p python3 env
source env/bin/activate
pip install -r requirements/dev.txt
```

```sh
<set your env variable>
python manage.py runserver 3001
```
